Deploy ke Railway:

1. Push repo ini ke GitHub tanpa file key.
2. Buat project baru di Railway dari repo GitHub tersebut.
3. Set Start Command ke:
   python run_all.py
4. Tambahkan environment variables:
   ACTIVE_BOTS=1,2,3,4,5
   KEY_1=isi api key bot 1
   KEY_2=isi api key bot 2
   KEY_3=isi api key bot 3
   KEY_4=isi api key bot 4
   KEY_5=isi api key bot 5

Catatan:
- script.py otomatis membaca KEY_n dari environment lebih dulu.
- Jika KEY_n tidak ada, bot fallback ke file key_n.txt.
- Untuk Railway, jangan upload key_*.txt ke GitHub.
